**Eye Map** is an open-source navigation application for people with visual impairments.

The voice AI assistant helps users reach their destinations by providing route guidance, bus numbers, and announcing stops.

## Alignment with DPG
This project is undergoing self-assessment as a Digital Public Good (DPG).

- **SDGs:** The project supports SDG [№10](https://sdgs.un.org/goals/goal10) and [№11](https://sdgs.un.org/goals/goal11).
